# Healthcare-Readmission-Analysis
## Overview

End-to-end healthcare analytics project using Excel, PostgreSQL, and Power BI. The project focuses on data cleaning, SQL-based KPI development, readmission analysis, cost analysis, and interactive dashboard creation for hospital operations and patient utilization insights.

## Project Objectives

* Clean and validate healthcare data
* Build a relational database in PostgreSQL
* Create healthcare KPIs and analytical views
* Analyze readmissions and treatment costs
* Develop an interactive Power BI dashboard


## Tools Used

* Excel
* PostgreSQL
* Power BI
* GitHub
